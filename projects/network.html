Proxy Server

In Python, I wrote a proxy server. This server could take HTTP 1.1 requests, both single and multi-line, and parse these requests. The proxy caches and pass requests that were correctly formatted using a buffer. Requests that did not follow the conventions both dictated by our professor and RFC would not be passed along, and the proxy server would respond with the appropriate error code.
